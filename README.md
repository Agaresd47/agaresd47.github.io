# Personal Site

This is a simple static personal website for an AI researcher / PhD-style academic profile.

## Files

- `index.html`: content and layout
- `styles.css`: visual design
- `script.js`: small footer enhancement

## Quick edits

Replace these placeholders in `index.html`:

- `Your Name`
- `Your University`
- `Prof. Advisor Name`
- `your.email@university.edu`
- `Project Title One/Two/Three`
- profile links like Scholar, LinkedIn, and CV

## Local preview

You can open `index.html` directly in a browser, or run a static server from this folder.

## Recommended GitHub repo name

- `agaresd47.github.io` if you want GitHub Pages at the root user URL
- `personal-site` if you just want a normal repository
